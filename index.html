basics

html
Hypertext markup-language is used to structure a webpage and its contents. An html document consists of a series of elements which you use to enclose or wrap around different parts of the content to make it appear or act a certain way.
anatomy of an element
<p class=""> test-content </p>
element name : p
opening tag : <p>
content : test-content
closing tag : </p>
attribute : class=""
empty elements
some elements have no content : <img src="" alt="">
block-level elements
a block-level element appears on a new line following the content that preceded it. Block-level elements are typically structural elements such as : <h1 - h6>, <p>, <ul or ol>, <nav>, <div>, <section>
inline elements
an inline element is typically contained within a block-level element. Inline elements will not cause a new line to appear. Examples of inline elements : <img>, <em>, <strong>, <span>, <code>, <var>, <samp>
anatomy of an html document
<!DOCTYPE html> --pre-amble
<html lang="en-US"> --root element

<head> --container for content that is not shown on the webpage
</head>

<body> --container for content that is shown onn the webpage

</header>

<nav> --a navigation bar will be defined in this element
</nav>

<main> --content that is specific & relevent to the webpage will go here
</main>

</footer>
</body>

</html>

what's in the <head>?

<meta>
metadata is data that essentially describes the webcontent data.
<meta charset="UTF-8">
--sets the character set of the document to UTF-8, which is a universal character set.
<meta name="author" content="author-name">
--sets metadata for the author equal to the author-name
<meta name="description" content="mdn-html notes">
--sets metadata for the description of the webpage equal to mdn-html notes. Adding keywords to the descriptions increases traction (SEO)
<title>mdn-html</title>
--sets the title of the webpage to mdn-html
<link rel="stylesheet" href="styles/style.css">
--links the style.css stylesheet to the html document
<link rel="icon" href="images/image.png" type="image/x-icon">
--sets the webpage logo for the favorites bar icon
<script src="scripts/main.js" defer></script>
--links main.js to the html document. defer : keyword that tells the browser to load the .js file after it has finished parsing the html document

</video>
